# [Project Name]

Property management system for [property description].

## Goals

- Organize property documents into a consistent, searchable structure
- Extract information quickly on demand
- Draft responses for tenants, agents, contractors, and insurers
- Maintain a chronological record of property events
- Track tasks and compliance obligations

## Non-Goals

- Not a web application
- Not a full accounting system
- Not replacing professional property management software
- No external integrations

## Requirements

### Must Have
- Document organizer (safe, proposes before moving)
- Timeline of property events (TIMELINE.md)
- Task tracking (TASKS.md)
- Status snapshots by domain (status/)
- Tax evidence folder (tax/)

### Should Have
- Info extractor agent
- Response writer agent
- Financial analyst / accountant agent
- Inbox auto-processing at session start
