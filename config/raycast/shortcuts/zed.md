|---------------------|-------------------|

| Next tab            | Alt+Shift+Right   |
| Split right         | Ctrl+Shift+R      |
| Split down          | Ctrl+Shift+D      |
| Close pane          | Ctrl+Shift+W      |
| Focus left pane     | Ctrl+Alt+Left     |
| Focus right pane    | Ctrl+Alt+Right    |
| Focus up pane       | Ctrl+Alt+Up       |
| Focus down pane     | Ctrl+Alt+Down     |
| Toggle sidebar      | Alt+Cmd+S         |
| AI agent            | Cmd+I             |
| Duplicate line      | Cmd+Shift+D       |
| Move line up        | Alt+Up            |
| Move line down      | Alt+Down          |
| Build               | Cmd+Shift+B       |
| Test                | Cmd+Shift+T       |
| Rerun last task     | Alt+R             |
